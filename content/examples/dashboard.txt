=== OVERVIEW ===
This example builds a multi-panel dashboard layout using higher-level components from
@ilovereact/core. It demonstrates NavPanel for navigation, Tabs for content switching,
Card for content containers, Table for data display, ProgressBar for metrics, and
Badge for status indicators. This is a realistic application layout that exercises
the flexbox system with nested containers.

=== CODE ===
---
import { useState } from 'react';
import {
  Box, Text, Card, Badge, Tabs, NavPanel, Table, ProgressBar, Divider,
} from '@ilovereact/core';

const NAV_SECTIONS = [
  {
    title: 'Overview',
    items: [
      { id: 'dashboard', label: 'Dashboard' },
      { id: 'analytics', label: 'Analytics' },
    ],
  },
  {
    title: 'Management',
    items: [
      { id: 'users', label: 'Users' },
      { id: 'settings', label: 'Settings' },
    ],
  },
];

const TABLE_COLUMNS = [
  { key: 'name', title: 'Name', width: 120 },
  { key: 'status', title: 'Status', width: 80, align: 'center' as const },
  { key: 'progress', title: 'Progress', width: 100 },
];

const TABLE_DATA = [
  { name: 'Project Alpha', status: 'Active', progress: 0.75 },
  { name: 'Project Beta', status: 'Review', progress: 0.45 },
  { name: 'Project Gamma', status: 'Done', progress: 1.0 },
  { name: 'Project Delta', status: 'Active', progress: 0.2 },
];

const TABS = [
  { id: 'overview', label: 'Overview' },
  { id: 'details', label: 'Details' },
  { id: 'history', label: 'History' },
];

function MetricCard({ title, value, change }: {
  title: string; value: string; change: string;
}) {
  const isPositive = change.startsWith('+');
  return (
    <Card style={{ flexGrow: 1 }}>
      <Text style={{ fontSize: 12, color: '#94a3b8' }}>{title}</Text>
      <Text style={{ fontSize: 28, color: '#f8fafc', marginTop: 4 }}>{value}</Text>
      <Text style={{
        fontSize: 12,
        color: isPositive ? '#22c55e' : '#ef4444',
        marginTop: 4,
      }}>
        {change}
      </Text>
    </Card>
  );
}

export default function App() {
  const [activeNav, setActiveNav] = useState('dashboard');
  const [activeTab, setActiveTab] = useState('overview');

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: '#0a0a12',
      flexDirection: 'row',
    }}>
      <NavPanel
        sections={NAV_SECTIONS}
        activeId={activeNav}
        onSelect={setActiveNav}
        header={
          <Text style={{ fontSize: 16, color: '#f8fafc', fontWeight: 'bold' }}>
            Dashboard
          </Text>
        }
        style={{ height: '100%' }}
      />
      <Box style={{ flexGrow: 1, padding: 16, gap: 16 }}>
        <Tabs tabs={TABS} activeId={activeTab} onSelect={setActiveTab} />
        <Box style={{ flexDirection: 'row', width: '100%', gap: 12 }}>
          <MetricCard title="Total Users" value="12,847" change="+12.5%" />
          <MetricCard title="Revenue" value="$48.2K" change="+8.3%" />
          <MetricCard title="Active Sessions" value="1,429" change="-2.1%" />
        </Box>
        <Card title="Projects" style={{ flexGrow: 1 }}>
          <Table
            columns={TABLE_COLUMNS}
            data={TABLE_DATA}
            striped
            style={{ width: '100%' }}
          />
        </Card>
        <Box style={{ flexDirection: 'row', width: '100%', gap: 12 }}>
          <Card title="CPU Usage" style={{ flexGrow: 1 }}>
            <ProgressBar value={0.67} color="#3b82f6" showLabel />
          </Card>
          <Card title="Memory" style={{ flexGrow: 1 }}>
            <ProgressBar value={0.42} color="#22c55e" showLabel />
          </Card>
          <Card title="Disk" style={{ flexGrow: 1 }}>
            <ProgressBar value={0.89} color="#ef4444" showLabel />
          </Card>
        </Box>
      </Box>
    </Box>
  );
}
---

=== EXPLANATION ===
1. The root layout is a row (flexDirection: 'row') splitting the viewport into a
   NavPanel on the left (fixed 180px width, default) and a content area (flexGrow: 1).

2. NavPanel renders a sidebar with section headers and selectable items. The
   activeId prop controls which item is highlighted. onSelect updates the state.

3. Tabs provide content switching within the main panel. The 'underline' variant
   (default) renders an underline indicator on the active tab.

4. MetricCard is a custom component wrapping Card. Each card uses flexGrow: 1 to
   distribute equally across the row. The row has width: '100%' and gap: 12.

5. Table renders a data grid with columns and rows. The striped prop alternates
   row backgrounds for readability.

6. ProgressBar shows a visual progress indicator. The value prop is 0-1 range.
   showLabel displays the percentage text.

7. The content area uses padding: 16 and gap: 16 to space all its children
   evenly. This avoids manual margin management on each child.
