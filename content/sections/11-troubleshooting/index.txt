=== OVERVIEW ===
This section covers the most common issues encountered when developing with iLoveReact.
The common errors page lists specific error symptoms and their solutions. The FAQ
addresses frequently asked questions about capabilities, limitations, and workflows.

If you are seeing unexpected behavior, start with the common errors page. Most issues
fall into a few categories: missing fontSize on Text nodes, missing dimensions on
containers, incorrect file editing (editing copies instead of source of truth), or
outdated runtime files.
