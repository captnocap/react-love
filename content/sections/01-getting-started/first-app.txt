=== OVERVIEW ===
This tutorial walks you through building an interactive counter application from
scratch. You will learn how to import primitives, manage state with React hooks, handle
user input with Pressable, and style your UI with flexbox properties. By the end you
will have a working app with increment and decrement buttons, styled with colors and
spacing.

=== EXAMPLES ===
Example 1: Step 1 - Create the Project
---
ilovereact init counter-app
cd counter-app
---
Platforms: All

Example 2: Step 2 - Basic Counter (src/App.tsx)
---
import { useState } from 'react';
import { Box, Text, Pressable } from '@ilovereact/core';

export default function App() {
  const [count, setCount] = useState(0);

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: '#0f172a',
      justifyContent: 'center',
      alignItems: 'center',
    }}>
      <Text style={{ fontSize: 48, color: '#f8fafc' }}>
        {`${count}`}
      </Text>
    </Box>
  );
}

-- Note: The counter displays but has no buttons yet.
-- The root Box uses width/height 100% to fill the viewport.
-- justifyContent and alignItems center the text in both axes.
-- The Text node has an explicit fontSize (required by the linter).
---

Example 3: Step 3 - Add Buttons
---
import { useState } from 'react';
import { Box, Text, Pressable } from '@ilovereact/core';

function CounterButton({ label, onPress }: { label: string; onPress: () => void }) {
  return (
    <Pressable onPress={onPress}>
      {({ pressed }) => (
        <Box style={{
          width: 60,
          height: 60,
          backgroundColor: pressed ? '#1d4ed8' : '#3b82f6',
          borderRadius: 12,
          justifyContent: 'center',
          alignItems: 'center',
        }}>
          <Text style={{ fontSize: 28, color: '#ffffff' }}>{label}</Text>
        </Box>
      )}
    </Pressable>
  );
}

export default function App() {
  const [count, setCount] = useState(0);

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: '#0f172a',
      justifyContent: 'center',
      alignItems: 'center',
      gap: 24,
    }}>
      <Text style={{ fontSize: 48, color: '#f8fafc' }}>
        {`${count}`}
      </Text>
      <Box style={{
        flexDirection: 'row',
        width: '100%',
        justifyContent: 'center',
        gap: 16,
      }}>
        <CounterButton label="-" onPress={() => setCount(c => c - 1)} />
        <CounterButton label="+" onPress={() => setCount(c => c + 1)} />
      </Box>
    </Box>
  );
}

-- Key concepts:
-- 1. Pressable provides press state via render props: ({ pressed }) => ...
-- 2. The row Box has width: '100%' so justifyContent: 'center' works.
--    (Row Boxes without explicit width ignore justifyContent.)
-- 3. gap: 16 adds space between the two buttons.
-- 4. Each button has fixed width/height for a square shape.
---
Platforms: All

=== CRITICAL RULES ===
- Every Text node MUST have an explicit fontSize.
- The root container MUST use width: '100%', height: '100%' -- not flexGrow: 1.
- Row Boxes need explicit width for justifyContent to work.
- Use template literals (backticks) for dynamic text: {`${count}`}, not mixed
  children like {count} which creates multiple text nodes.
- Always run `ilovereact lint` after modifying components.
