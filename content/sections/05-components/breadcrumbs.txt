=== OVERVIEW ===
Breadcrumbs renders a horizontal trail of navigation items separated by a configurable separator character. Each item except the last is clickable via Pressable and triggers the `onSelect` callback with the item's id. The last item in the trail is displayed as bold text, indicating the current location.

The component is built with Box, Text, and Pressable primitives, making it work identically across all targets.

=== API / SYNTAX ===
Import: import { Breadcrumbs } from '@ilovereact/core';

BreadcrumbItem type:
```ts
interface BreadcrumbItem {
  id: string;
  label: string;
}
```

Usage:
```tsx
<Breadcrumbs
  items={[
    { id: 'home', label: 'Home' },
    { id: 'settings', label: 'Settings' },
    { id: 'profile', label: 'Profile' },
  ]}
  onSelect={(id) => navigate(id)}
/>
```

Props:

| Prop      | Type                       | Default | Description                             |
|-----------|----------------------------|---------|-----------------------------------------|
| items     | BreadcrumbItem[]           | (required)| Array of breadcrumb path items        |
| onSelect  | (id: string) => void       | -       | Called when a non-last item is clicked  |
| separator | string                     | '/'     | Separator character between items       |
| style     | Style                      | -       | Container style overrides               |

=== PLATFORM NOTES ===
- Breadcrumbs uses Pressable for clickable items, inheriting hover behavior. Non-last items have `opacity: 0.7` by default, increasing to `opacity: 1` on hover.
- The container uses `flexDirection: 'row'`, `width: '100%'`, `alignItems: 'center'`, and `gap: 6`.
- Non-last item text: `color: '#64748b'`, `fontSize: 11`.
- Last item text: `color: '#e2e8f0'`, `fontSize: 11`, `fontWeight: 'bold'`.
- Separator text: `color: '#334155'`, `fontSize: 11`.

=== CRITICAL RULES ===
- The `items` prop is required. An empty array renders nothing.
- The last item in the array is always rendered as non-clickable bold text. It does not trigger `onSelect`.
- Only non-last items are wrapped in Pressable. The last item is a plain Text element.
- Breadcrumbs renders in a row layout. Ensure the parent container has enough width for the trail, or items will wrap depending on the parent's flexWrap setting.
