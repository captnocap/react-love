=== OVERVIEW ===
Toolbar renders a horizontal row of action items with optional dividers for visual
grouping. Each item is a labeled pressable button. Dividers are thin vertical lines
that separate logical groups.

Items are provided as an array of ToolbarEntry objects — either { type: 'item' } with
an id, label, and optional disabled flag, or { type: 'divider' } for visual separation.

=== API / SYNTAX ===
Import:
  import { Toolbar } from '@ilovereact/core'

Types:
  type ToolbarEntry =
    | { type: 'item'; id: string; label: string; disabled?: boolean }
    | { type: 'divider' }

  interface ToolbarProps {
    items: ToolbarEntry[];
    onSelect?: (id: string) => void;
    style?: Style;
  }

Props:
  items       ToolbarEntry[]         Required. Array of items and dividers.
  onSelect    (id: string) => void   Called when an enabled item is pressed. Receives the item id.
  style       Style                  Merged onto the root container.

ToolbarEntry (item):
  type        'item'                 Discriminant.
  id          string                 Unique identifier passed to onSelect.
  label       string                 Display text.
  disabled    boolean                Optional. Greys out and disables press. Default false.

ToolbarEntry (divider):
  type        'divider'              Renders a vertical separator line.

  Dividers render as thin Box elements with backgroundColor.
Web:
  Same rendering as Love2D — Pressable maps to interactive div elements.
