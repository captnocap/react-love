=== OVERVIEW ===
The Style interface defines all visual and layout properties for iLoveReact nodes. It is
passed via the `style` prop on Box, Text, Image, and other components. All properties
are optional. Properties that are not set use their default values (typically 0, 'column',
or undefined).

Colors can be CSS hex strings ("#ff0000", "#ff000080" with alpha), CSS function notation
("rgba(255, 0, 0, 0.5)"), or Love2D-style number arrays [r, g, b, a] where each value
is 0-1.

=== API / SYNTAX ===

--- SIZING ---

width?: number | string
  Width of the node. Number = pixels. String = percentage ("50%"), viewport ("10vw").
  Default: Inherits parent's width for non-text nodes. Text nodes auto-size.

height?: number | string
  Height of the node. Number = pixels. String = percentage, viewport ("5vh").
  Default: Auto-sizes to content.

minWidth?: number | string
  Minimum width constraint. Applied after flex distribution.
  Default: none

minHeight?: number | string
  Minimum height constraint. Applied after flex distribution.
  Default: none

maxWidth?: number | string
  Maximum width constraint. Applied after flex distribution.
  Default: none

maxHeight?: number | string
  Maximum height constraint. Applied after flex distribution.
  Default: none

aspectRatio?: number
  Width-to-height ratio. When one dimension is set, the other is computed.
  Example: aspectRatio: 16/9 means width = height * (16/9).
  Default: none

--- FLEXBOX ---

display?: 'flex' | 'none'
  'flex': Normal layout participation (default).
  'none': Node and all descendants skipped entirely from layout and painting.
  Default: 'flex'

flexDirection?: 'row' | 'column'
  Main axis direction. 'column' stacks children vertically. 'row' lays out horizontally.
  Default: 'column'

flexWrap?: 'nowrap' | 'wrap'
  'nowrap': All children on one line (default). 'wrap': Children wrap to next line.
  Default: 'nowrap'

justifyContent?: 'start' | 'center' | 'end' | 'space-between' | 'space-around' | 'space-evenly'
  Distribution of children along the main axis.
  Only effective when the container has a known main-axis size.
  Default: 'start'

alignItems?: 'start' | 'center' | 'end' | 'stretch'
  Alignment of children along the cross axis.
  'stretch': Children expand to fill the cross axis (unless they have explicit size).
  Default: 'stretch'

alignSelf?: 'auto' | 'start' | 'center' | 'end' | 'stretch'
  Per-child override of the parent's alignItems.
  'auto': Use parent's alignItems value.
  Default: 'auto'

flexGrow?: number
  Proportion of positive free space to absorb. Only works when the parent has
  definite main-axis size and there is leftover space after placing all children.
  Default: 0

flexShrink?: number
  Proportion of negative overflow to absorb. Activated when children overflow
  the parent's main axis. CSS default is 1 (items shrink proportionally).
  Default: 1

flexBasis?: number | string | 'auto'
  Initial main-axis size before flex distribution. Takes priority over width/height.
  'auto': Falls back to width (row) or height (column).
  Default: 'auto'

gap?: number | string
  Space between children on the main axis AND between lines when wrapping.
  Applied between items only (not before the first or after the last).
  Default: 0

--- SPACING ---

padding?: number | string
  Inner space on all four sides. Percentage resolves against the node's width.
  Default: 0

paddingLeft?: number | string
  Left inner space. Overrides padding for the left side.

paddingRight?: number | string
  Right inner space. Overrides padding for the right side.

paddingTop?: number | string
  Top inner space. Overrides padding for the top side.

paddingBottom?: number | string
  Bottom inner space. Overrides padding for the bottom side.

margin?: number | string
  Outer space on all four sides. Percentage resolves against the parent's width.
  Default: 0

marginLeft?: number | string
  Left outer space. Overrides margin for the left side.

marginRight?: number | string
  Right outer space. Overrides margin for the right side.

marginTop?: number | string
  Top outer space. Overrides margin for the top side.

marginBottom?: number | string
  Bottom outer space. Overrides margin for the bottom side.

--- VISUAL ---

backgroundColor?: Color
  Fill color for Box/View nodes. Hex string, rgba string, or [r,g,b,a] array.
  Default: transparent

borderRadius?: number
  Uniform corner radius in pixels. Applied to all four corners.
  Default: 0

borderTopLeftRadius?: number
  Per-corner override. Takes precedence over borderRadius.

borderTopRightRadius?: number
  Per-corner override. Takes precedence over borderRadius.

borderBottomLeftRadius?: number
  Per-corner override. Takes precedence over borderRadius.

borderBottomRightRadius?: number
  Per-corner override. Takes precedence over borderRadius.

borderWidth?: number
  Uniform border stroke width on all sides.
  Default: 0

borderTopWidth?: number
  Per-side border width. Overrides borderWidth for the top side.

borderRightWidth?: number
  Per-side border width. Overrides borderWidth for the right side.

borderBottomWidth?: number
  Per-side border width. Overrides borderWidth for the bottom side.

borderLeftWidth?: number
  Per-side border width. Overrides borderWidth for the left side.

borderColor?: Color
  Uniform border color on all sides.
  Default: gray (0.5, 0.5, 0.5, 1)

borderTopColor?: Color
  Per-side border color. Overrides borderColor for the top side.

borderRightColor?: Color
  Per-side border color. Overrides borderColor for the right side.

borderBottomColor?: Color
  Per-side border color. Overrides borderColor for the bottom side.

borderLeftColor?: Color
  Per-side border color. Overrides borderColor for the left side.

overflow?: 'visible' | 'hidden' | 'scroll'
  'visible': Content can extend beyond bounds (default).
  'hidden': Content clipped to bounds (stencil for rounded, scissor for rectangular).
  'scroll': Scrollable container with clipping and scroll indicators.
  Default: 'visible'

opacity?: number
  Node opacity (0-1). Multiplies with inherited opacity from parent chain.
  0 = fully transparent. 1 = fully opaque.
  Default: 1

zIndex?: number
  Paint order within the parent's children. Higher values paint on top.
  Does NOT affect layout, only paint order.
  Default: 0

scrollX?: number
  Horizontal scroll position for scroll containers (controlled mode).

scrollY?: number
  Vertical scroll position for scroll containers (controlled mode).

--- SHADOW ---

shadowColor?: Color
  Box shadow color. Shadow only renders when shadowBlur > 0.

shadowOffsetX?: number
  Horizontal shadow offset in pixels.
  Default: 0

shadowOffsetY?: number
  Vertical shadow offset in pixels.
  Default: 0

shadowBlur?: number
  Shadow blur radius. Simulated by rendering multiple fading rectangles.
  Capped at 10 blur steps for performance.
  Default: 0

--- GRADIENT ---

backgroundGradient?: { direction, colors }
  Linear gradient background. Takes precedence over backgroundColor.
  direction: 'horizontal' | 'vertical' | 'diagonal'
  colors: [Color, Color] -- start and end colors
  Rendered via Love2D mesh API on Love2D, CSS linear-gradient on web.

--- TRANSFORM ---

transform?: { translateX?, translateY?, rotate?, scaleX?, scaleY?, skewX?, skewY?, originX?, originY? }
  Visual transform. Does NOT affect layout or hit testing.
  translateX/Y: Pixel offset.
  rotate: Degrees (not radians).
  scaleX/Y: Scale factor (1 = no scale).
  skewX/Y: Skew angle in degrees.
  originX/Y: Transform origin as 0-1 fraction (0.5 = center, default).

--- TEXT ---

color?: Color
  Text color. Applied to Text and __TEXT__ nodes.
  Inherited by __TEXT__ children from parent Text nodes.
  Default: white (1, 1, 1, 1)

fontSize?: number
  Font size in pixels. REQUIRED on all Text nodes (enforced by linter).
  Inherited by __TEXT__ children from parent Text nodes.
  Default: 14 (but the linter requires explicit values)

fontFamily?: string
  Path to a .ttf or .otf font file. Relative to the Love2D filesystem root.
  Inherited by __TEXT__ children from parent Text nodes.
  Default: Love2D default font

fontWeight?: 'normal' | 'bold' | number
  Font weight. 'bold' triggers faux-bold rendering (double draw with offset).
  Default: 'normal'

textAlign?: 'left' | 'center' | 'right'
  Horizontal text alignment within the node's computed width.
  Default: 'left'

textOverflow?: 'clip' | 'ellipsis'
  How to handle text that overflows the node's width.
  'ellipsis': Truncate with "..." at the end.
  Default: 'clip'

textDecorationLine?: 'none' | 'underline' | 'line-through'
  Text decoration line style.
  Default: 'none'

lineHeight?: number
  Line height in pixels. Overrides the font's natural line height.
  When set, text is rendered line-by-line with this spacing.
  Default: Font's natural line height

letterSpacing?: number
  Extra space between characters in pixels.
  WARNING: Activates character-by-character rendering. Expensive on large text.
  Default: 0

--- TEXT SHADOW ---

textShadowColor?: Color
  Text shadow color. Shadow only renders when this is set.

textShadowOffsetX?: number
  Horizontal text shadow offset.
  Default: 0

textShadowOffsetY?: number
  Vertical text shadow offset.
  Default: 0

--- IMAGE ---

objectFit?: 'fill' | 'contain' | 'cover' | 'none'
  How the image scales within its bounds.
  'fill': Stretch to fill (default). 'contain': Fit inside, letterbox.
  'cover': Fill, crop overflow. 'none': No scaling, center.
  Default: 'fill'

--- POSITIONING ---

position?: 'relative' | 'absolute'
  'relative': Normal flow (default).
  'absolute': Positioned relative to the nearest positioned ancestor.
  Default: 'relative'

top?: number | string
  Offset from the top edge (absolute positioning).

bottom?: number | string
  Offset from the bottom edge (absolute positioning).

left?: number | string
  Offset from the left edge (absolute positioning).

right?: number | string
  Offset from the right edge (absolute positioning).

--- VISIBILITY ---

visibility?: 'visible' | 'hidden'
  'visible': Normal rendering (default).
  'hidden': Node is not painted, but still occupies layout space.
  Unlike display:'none', children of a hidden node still paint.
  Default: 'visible'

--- OUTLINE ---

outlineColor?: Color
  Outline color. Drawn outside the border box.
  Default: white (1, 1, 1, 1) or currentColor on web

outlineWidth?: number
  Outline stroke width.
  Default: 0

outlineOffset?: number
  Gap between the border box and the outline.
  Default: 0

--- TRANSITION ---

transition?: { [property: string]: { duration?, easing?, delay? } }
  CSS-style transitions. Lua-side: JS declares transition targets, Lua interpolates.
  Web-side: Mapped directly to CSS transition property.

  Per-property: transition: { backgroundColor: { duration: 300, easing: 'easeInOut' } }
  All properties: transition: { all: { duration: 300 } }

  duration: number in milliseconds (default: 300)
  easing: 'linear' | 'easeIn' | 'easeOut' | 'easeInOut' | 'bounce' | 'elastic' (default: 'easeInOut')
  delay: number in milliseconds (default: 0)

--- ANIMATION ---

animation?: { keyframes, duration?, easing?, iterations?, direction?, fillMode?, delay?, playState? }
  CSS-style keyframe animations. Runs on the Lua side.

  keyframes: { [percentage: number]: Partial<Style> }
    Percentage keys from 0 to 100. Each value is a partial Style object.
    Example: { 0: { opacity: 0 }, 100: { opacity: 1 } }

  duration: number in milliseconds (default: 300)
  easing: string easing function name (default: 'linear')
  iterations: number, -1 = infinite (default: 1)
  direction: 'normal' | 'reverse' | 'alternate' | 'alternate-reverse' (default: 'normal')
  fillMode: 'none' | 'forwards' | 'backwards' | 'both' (default: 'none')
  delay: number in milliseconds (default: 0)
  playState: 'running' | 'paused' (default: 'running')
