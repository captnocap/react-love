=== OVERVIEW ===
This section contains the complete API reference for iLoveReact. The style properties
document lists every property available in the Style interface with types, valid values,
and descriptions. The types document provides the full TypeScript type definitions for
all components, events, and animation interfaces.

These references are derived directly from the source code in packages/shared/src/types.ts
and related files.
