=== OVERVIEW ===
This reference lists all TypeScript types exported from @ilovereact/core (packages/shared).
Types are organized by category: core types, primitive component props, higher-level
component props, animation types, and utility types. All types are importable from
'@ilovereact/core'.

=== API / SYNTAX ===

--- CORE TYPES ---

Color = string | [number, number, number, number?]
  CSS string ("#ff0000", "rgba(255,0,0,0.5)") or Love2D array [r, g, b, a] (0-1 range).
  The alpha channel is optional and defaults to 1.

Style
  Full style interface. See style-properties.txt for every property.
  All properties are optional.

RendererMode = 'web' | 'native'
  Which rendering path the primitives use.
  'web': DOM elements (div/span/img).
  'native': Host elements for react-reconciler (View/Text/Image).
